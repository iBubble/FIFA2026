# 4. “绿茵深邃暗夜流光”设计系统与视觉美学

为了在视觉上震撼用户，我们决定为世界杯赛事网站定制一套高度契合现代体育竞技和未来科技感的“绿茵深邃暗夜流光 (Pitch Temple Dark Mode)”设计系统：

1. **色彩规范**：
   - 背景色：球场夜空渐变黑绿（`#040D0A` 至 `#0B1A15`）。
   - 核心主题色（霓虹绿）：`#00FF9D`，代表生机勃勃的草坪与未来感，用于出线、高光和成功交互状态。
   - 辅助高光色（圣殿金）：`#F3C623`，代表顶级荣誉，用于金靴射手榜、冠军王座等。
   - 文本与边框：钛白（`#F3F4F6`）、冰蓝（`#E0F2FE`）与半透明灰色。
2. **材质规范**：
   - 广泛使用毛玻璃质感（Glassmorphism，利用 `backdrop-blur-md` 和极细的微透白边框 `rgba(255,255,255,0.05)`），使得卡片富含空气感和悬浮感。
3. **动效规范**：
   - 按钮、输入框和国旗卡片具有平滑的 `transition` 和悬浮微缩放效果。
   - 在晋级模拟沙盘中，点击晋级后的获胜路径以荧光绿色流光动画（Glow Line Path）连线形式流畅过渡。

这一风格兼具高质感和未来感，能够有效提升用户的探索与操作欲望。
